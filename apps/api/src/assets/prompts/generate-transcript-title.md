# Generate Transcript Title

You are a title generation specialist. Your task is to create a concise, descriptive title for the following transcript.

## Transcript Content
{{TRANSCRIPT_CONTENT}}

## Requirements
1. **Length**: Keep the title between 3-10 words
2. **Clarity**: Make it immediately clear what the content is about
3. **Specificity**: Be specific rather than generic (avoid titles like "Discussion" or "Conversation")
4. **Format**: Use title case (capitalize major words)
5. **No Quotes**: Don't wrap the title in quotes or add any special characters

## Examples of Good Titles
- Building a Content Creation Pipeline
- Deep Learning Model Optimization Techniques
- Customer Success Strategy Meeting Notes
- React Performance Best Practices Discussion
- Q4 Marketing Campaign Planning Session

## Examples of Bad Titles
- Meeting Notes
- Discussion
- Audio Recording
- Transcript
- Conversation About Stuff

Generate a single title that best captures the essence of the transcript content. Return only the title text, nothing else.
